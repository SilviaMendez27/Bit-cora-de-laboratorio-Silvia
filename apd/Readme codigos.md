# 💻 Procesamiento de Imágenes y Modelos de Clasificación con Python
#### Este repositorio reúne una serie de proyectos desarrollados en Python enfocados en el procesamiento digital de imágenes y la aplicación de modelos de clasificación supervisada. Incluye técnicas de mejora de contraste, umbralización, detección de bordes, segmentación básica y análisis de datos mediante algoritmos de machine learning.

### 🎯 Objetivo
#### Desarrollar e integrar diferentes herramientas computacionales para el análisis de imágenes y datos, utilizando técnicas de visión por computadora y aprendizaje automático en Python.

### 📂 Contenido del repositorio
#### Este repositorio contiene los siguientes proyectos:
#### - **app.py**  
#### Aplicación interactiva desarrollada en Streamlit para visualización básica y uso de controles dinámicos.
#### - **canny+clahe+thres.py**  
#### Script de procesamiento de imágenes que combina mejora de contraste con CLAHE, umbralización y detección de bordes mediante Canny.
#### - **ejercicio integrador.py**  
#### Flujo completo de procesamiento de imágenes que integra extracción de canal de color, CLAHE, umbral adaptativo, inversión, operaciones morfológicas y detección de bordes.
#### - **THeshold.py**  
#### Comparación de diferentes métodos de umbralización, incluyendo threshold global, adaptive thresholding y Otsu.
#### - **ejercicio-taller de regresión logística, knn y svm.ipynb**  
#### Notebook de clasificación supervisada con modelos de regresión logística, KNN y SVM.

### ⚙️ Requisitos
#### Para ejecutar los proyectos de este repositorio se recomienda contar con:

#### - Python 3.x
#### - OpenCV
#### - NumPy
#### - Matplotlib
#### - Pandas
#### - Streamlit
#### - Scikit-learn
#### - Pillow
